## Option
---
Show
```
todogotcha -h
```

``` -root="<Specify search root directory>" ```
``` -filetype="<Target file type list>" ```
``` -keyword="<Gather target word>" ```

Example
```
todogotcha -root="../" \
          -filetype="go" \
          -keyword="TODO:"
```
