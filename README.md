# rgr

For learn.

rgr is cli tools for find target words in many files.

## Usage

```sh
# show help
rgr -help

# search "func" in specific files and directories.
rgr "func" file.txt dir/

# search recursively from current directory.
rgr "func"
```

## Installation

```sh
go get github.com/yaeshimo/rgr
```
