# rgr

For learn.

rgr is cli tool for find target words in many files.
Available in utf8.

## Usage

```sh
# show help
rgr -help

# search "func" in specific files and directories.
rgr "func" file.txt dir/

# search "func" from current directory
rgr "func"
```

## Installation

```sh
go get github.com/yaeshimo/rgr
```
